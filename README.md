# 전화틈틈

전화틈틈은 정해둔 번호 목록을 기준으로 자동발신 예약을 만들고, 통화 기록과 반려폰 산책 기능을 함께 관리하는 안드로이드 APK입니다.


# 최신 공식 배포 버전

공식 배포 버전: v1.0.4
배포일: 2026-07-05
GitHub Releases에는 위 APK를 최신 릴리스 파일로 올려 배포합니다.


## 주요 기능

- 명의/전화번호 목록 관리
- 랜덤 자동발신 예약 생성
- 예약 간격, 예약 시간대, 자동종료 시간 설정
- 통화 기록과 월별 통계 확인
- 반려폰 산책 기능(기준반경 이탈시 자동 기지국 전화)
- 기지국/위치 기반 산책 기록 확인
- 새 버전 업데이트 안내


## 설치 방법

1. 아래 Releases에서 최신 APK를 받습니다.
2. 휴대폰에서 APK 파일을 엽니다.
3. 설치 중 "알 수 없는 앱 설치" 허용이 필요하면 허용합니다.
4. 앱 실행 후 필요한 권한을 허용합니다.

## v1.0.0 변경 내용

- 자동수신은 의무전화 수신 체크 번호와 반려폰 산책 짝꿍번호만 받도록 정리했습니다.
- 반려폰 산책 자동발신은 오늘 실제 짝꿍번호 통화기록이 없으면 계속 재시도하도록 바꿨습니다.
- 짝꿍번호 입력 후 다른 곳을 눌렀을 때 저장값과 화면 표시가 바로 맞도록 다듬었습니다.

## 주의사항

이 앱은 APK 직접 배포용입니다. 자동발신, 자동수신, 통화기록, 위치 권한을 사용하므로 설치와 권한 허용은 사용자가 직접 확인하고 진행해야 합니다.
