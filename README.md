# 전화틈틈

전화틈틈은 정해둔 번호 목록을 기준으로 자동발신 예약을 만들고, 통화 기록과 반려폰 산책 기능을 함께 관리하는 안드로이드 APK입니다.

## 주요 기능

- 명의/전화번호 목록 관리
- 랜덤 자동발신 예약 생성
- 예약 간격, 예약 시간대, 자동종료 시간 설정
- 통화 기록과 월별 통계 확인
- 반려폰 산책 기능
- 기지국/위치 기반 산책 기록 확인
- 새 버전 업데이트 안내
- 커피후원 링크

## 설치 방법

1. 아래 Releases에서 최신 APK를 받습니다.
2. 휴대폰에서 APK 파일을 엽니다.
3. 설치 중 "알 수 없는 앱 설치" 허용이 필요하면 허용합니다.
4. 앱 실행 후 필요한 권한을 허용합니다.

최신 버전 다운로드:

https://github.com/babypepper/call-teum/releases/latest

- 자동업데이트 확인 주소를 `babypepper/call-teum` 저장소로 변경했습니다.

## 업데이트 안내 방식

앱은 저장소의 `update.txt` 파일을 확인해서 새 버전이 있으면 업데이트 안내를 띄웁니다.

새 버전을 올릴 때는 다음 두 가지를 함께 수정합니다.

- Releases에 새 APK 업로드
- `update.txt`의 `versionCode`, `versionName`, `apkUrl` 수정

## 주의사항

이 앱은 APK 직접 배포용입니다. 자동발신, 자동수신, 통화기록, 위치 권한을 사용하므로 설치와 권한 허용은 사용자가 직접 확인하고 진행해야 합니다.
